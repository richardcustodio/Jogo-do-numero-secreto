# **Jogo do Número Secreto / Secret Number Game**

## 🕹️ **Como funciona? / How does it work?**
O jogador precisará adivinhar um número gerado aleatoriamente de 1 a 10. Para vencer, o jogador deve acertar o número gerado.  
The player needs to guess a randomly generated number between 1 and 10. To win, the player must correctly guess the generated number.

---

## 🛠️ **Tecnologias Utilizadas / Technologies Used**
- **HTML5**: Estrutura do jogo.  
  **HTML5**: Game structure.
- **CSS3**: Estilização da interface.  
  **CSS3**: Interface styling.
- **JavaScript**: Lógica de geração de número aleatório e verificação do palpite.  
  **JavaScript**: Random number generation and guess-checking logic.

## 🌟 **Funcionalidades / Features**
- Geração de um número aleatório entre 1 e 10.  
  Generates a random number between 1 and 10.
- Verificação automática do palpite do jogador.  
  Automatically checks the player's guess.
- Mensagens de vitória ou erro para orientar o jogador.  
  Displays victory or error messages to guide the player.
- Reinício do jogo após cada tentativa concluída.  
  Restarts the game after each completed attempt.
